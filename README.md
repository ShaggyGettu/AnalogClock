# AnalogClock
 Analog clock widget for flutter
 
 # 12 Hour mode with minutes and second picker:
 
 ```dart
 DateTime dateTime1 = await AnalogClock.showAnalogClock(
            context: context,
            dateTime: dateTime,
            hour12MinuteSecond: true,
          );

```
